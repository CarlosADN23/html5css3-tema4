HTML5

Veremos los nuevos elementos estructurales, así como los nuevos elementos existentes para formularios, audio y vídeo. El nuevo elemento canvas nos abre un mundo de posibilidades en el desarrollo de videojuegos web.

CSS3

Mejoraremos nuestros diseños web utilizando las nuevas características de CSS3: efectos, transformaciones, animaciones, etc. Con CSS3 nuestros diseños serán más simples, pero no menos impactantes.

Javascript

Junto con HTML5 aparecen nuevas API javascript que nos aportan multitud de facilidades y posibilidades en el desarrollo de nuestros proyectos web. Con las nuevas API javascript nuestras manos dejarán de estar atadas.
